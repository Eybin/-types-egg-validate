# Installation
> `npm install --save @eybin-types/egg-validate`

# Usage

```json
 // {app_root}/tsconfig.json
 ...
 "compilerOptions": {
    ...
    "typeRoots": [
        "node_modules/@types",
        "node_modules/@eybin-types" // specify the directory for type definition file to be included
    ],
    ...
},
...
```

# Summary
